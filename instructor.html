LogiCola Instructor Report — verify and review student score files (.lgc)

Drop student score files here — or

You can select a whole folder's worth at once. Nothing is uploaded anywhere — analysis happens in your browser.

How to read the integrity badges. verified — the file is byte-for-byte what LogiCola exported, and its internal activity chain is intact. altered — the file's signature or activity chain doesn't check out: it was edited after export (or corrupted in transit; ask the student to re-export before assuming the worst). unsigned — an old-format export with no activity log; it shows levels only and can't be verified.

What this can and cannot tell you. LogiCola runs entirely on the student's computer, so a determined student who reads the source code could fabricate a file. These checks are designed so that fabricating a convincing file — hundreds of correctly-answered, correctly-timed, hash-chained logic problems — is more work than doing the exercises. A verified badge plus unremarkable timing statistics is strong evidence of real effort; any ⛔ or ⚠️ flag is a reason to look closer, not automatic proof of cheating.
